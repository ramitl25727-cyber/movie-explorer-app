# 🎬 Movie Explorer App

## 📌 Description

This project is a web application that fetches movie data from a public API and displays it in a user-friendly interface. Users will be able to search, filter, and sort movies.

## 🚀 Features

- Search movies by name
- Filter movies by rating or genre
- Sort movies by popularity
- Responsive UI design

## 🛠 Tech Stack

- HTML
- CSS
- JavaScript (Fetch API)

## 🌐 API Used

TMDB (The Movie Database API)

## ▶️ How to Run

1. Clone the repository
2. Open index.html in browser
